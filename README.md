# Carbon Weighted Traffic Control — Project Website

This is a small React + Vite site scaffold to showcase the Carbon Weighted Traffic Control senior project.

Quick start (PowerShell):

```powershell
cd "c:/Documents/Projects/Senior-Project-Website"
# Install dependencies
npm install
# Start dev server
npm run dev
```

Notable files:
- `index.html` — app entry
- `src/main.jsx` — React entry
- `src/App.jsx` — main UI with team, gallery, and placeholders
- `src/styles.css` — project styles
